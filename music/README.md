# 本地音乐目录

请将你的背景音乐放在这个文件夹中，支持的格式：
- MP3
- WAV
- OGG
- M4A

## 目录结构示例：
```
music/
├── jingle-bells.mp3
├── silent-night.mp3
├── we-wish-you.mp3
├── ambient/
│   ├── winter-sounds.wav
│   └── snow-falling.mp3
└── classical/
    ├── christmas-symphony.m4a
    └── holiday-piano.mp3
```

## 使用方法：
1. 将音乐文件复制到此文件夹
2. 在 music.json 中配置对应的路径
3. 刷新页面即可听到背景音乐

## 推荐的圣诞音乐：
- Jingle Bells (铃儿响叮当)
- Silent Night (平安夜)
- We Wish You a Merry Christmas (祝你圣诞快乐)
- White Christmas (白色圣诞)
- Let It Snow! Let It Snow! Let It Snow!
- The Christmas Song (圣诞歌)

## 注意事项：
- 建议音乐文件大小不超过10MB以获得最佳加载性能
- 支持子文件夹组织音乐
- 文件名建议使用英文和数字，避免特殊字符
- 音量建议设置在0.3-0.7之间，避免过于突兀
- 可以设置循环播放作为背景音乐
