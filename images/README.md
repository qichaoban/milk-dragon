# 本地图片目录

请将你的照片放在这个文件夹中，支持的格式：
- JPG/JPEG
- PNG
- WebP
- GIF

## 目录结构示例：
```
images/
├── photo1.jpg
├── photo2.png
├── family/
│   ├── christmas2023.jpg
│   └── vacation.png
└── memories/
    ├── birthday.jpg
    └── wedding.png
```

## 使用方法：
1. 将照片复制到此文件夹
2. 在 images.json 中配置对应的路径
3. 刷新页面即可看到照片在3D场景中显示

## 注意事项：
- 建议图片大小不超过2MB以获得最佳性能
- 支持子文件夹组织照片
- 文件名建议使用英文和数字，避免特殊字符
